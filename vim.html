Vim Script

Vim Script — динамический сценарный язык, встроенный в тестовый редактор Vim. Сам по себе язык достаточно стандартный, однако требует знания тонкостей и команд редактора, что наделяет Vim Script высоким порогом вхождения. Помимо особенностей, что язык получает от Vim, у него есть и собственные — в Vim Script существует восемь областей видимости переменных, а для хранения данных могут использоваться не только переменные, но и регистры и опции редактора.
